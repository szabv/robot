## Robot Controller

This is to demonstrate some front end fun. 
Uses React and Styled-Components to move a small robot icon around a grid. 

To get started:

Install the dependencies with yarn or npm. 
```sh
yarn
```

Start a local development server: 
```sh
yarn start
```

Run the unit tests: 
```sh
yarn test
```

Build the project: 
```sh
yarn build
```
